# Modules

`unagent` is organized into focused modules, each exported separately for tree-shaking.

## Available Modules

| Module | Import | Description |
|--------|--------|-------------|
| [env](/modules/env) | `unagent/env` | Agent detection, config paths, XDG |
| [skill](/modules/skill) | `unagent/skill` | Skill discovery and markdown parsing |
| [source](/modules/source) | `unagent/source` | URL and source string parsing |
| [git](/modules/git) | `unagent/git` | Clone repos, git operations |
| [link](/modules/link) | `unagent/link` | Copy directories, symlinks |
| [lock](/modules/lock) | `unagent/lock` | Skill lockfile management |
| [utils](/modules/utils) | `unagent/utils` | ANSI colors, formatting, paths |

## Full Import

Import everything from the main entry:

```ts
import { detectCurrentAgent, discoverSkills, parseSource } from 'unagent'
```

## Selective Import

Import only what you need for smaller bundles:

```ts
import { detectCurrentAgent } from 'unagent/env'
import { discoverSkills } from 'unagent/skill'
import { parseSource } from 'unagent/source'
```
